# Adam Assistant для Minecraft 1.21.11 Fabric

Мод добавляет серверного чат-ассистента "Адам". Игрок пишет сообщение в чат, начинающееся с `адам`, и мод перехватывает его до отправки обычному чату.

Примеры:

```text
адам дай бревно акации
адам дай 64 алмаза
адам дай стак дубовых досок
адам дай minecraft:acacia_log
адам выполни /time set day
/adam дай бревно акации
```

## Что уже реализовано

- Триггер по обычному чату: сообщение должно начинаться с `адам`.
- Альтернативная команда: `/adam <запрос>`.
- Выдача предметов по русским фразам и по ID предмета.
- Примитивный NLP-парсер для количества: `64`, `стак`, `2 стака`, `полстака`, `один`, `два`, `три` и т.д.
- Выполнение обычных Minecraft-команд через фразу `адам выполни /...` с правами самого игрока.

## Важное ограничение

Это не подключение к ChatGPT/OpenAI. Внутри сейчас локальный rule-based парсер: он быстро работает без интернета и API-ключей. Если нужен настоящий LLM-парсер, добавляй HTTP-клиент и отправляй текст запроса в свою модель, а результат превращай в безопасное действие: `give`, `command`, `reply`.

## Сборка

Нужно:

- Java 21.
- Gradle 9.2+ или IDE, которая умеет открывать Gradle-проекты.
- Fabric Loader 0.18.1+.
- Fabric API для Minecraft 1.21.11.

Сборка:

```bash
gradle build
```

Готовый `.jar` появится в:

```text
build/libs/adam-assistant-1.0.0.jar
```

Положи этот `.jar` и Fabric API в папку `mods` клиента/сервера Fabric 1.21.11.

## Безопасность на сервере

По умолчанию выдача предметов разрешена всем игрокам. Для публичного сервера лучше открыть файл:

```text
src/main/java/dev/adamassistant/AdamAssistant.java
```

и поменять:

```java
private static final boolean REQUIRE_OP_FOR_GIVE = false;
```

на:

```java
private static final boolean REQUIRE_OP_FOR_GIVE = true;
```

Тогда `адам дай ...` будет работать только у OP / игроков с permission level 2+.

## Как добавить свои русские названия предметов

Открой:

```text
src/main/java/dev/adamassistant/ItemResolver.java
```

и добавь строку в `registerBasicItems()`:

```java
register("minecraft:elytra", "элитры", "надкрылья");
```

Формат:

```java
register("minecraft:item_id", "русское название 1", "русское название 2");
```
